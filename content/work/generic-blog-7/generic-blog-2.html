Introduction

The Tufte handout style is a style that Edward Tufte uses in his books and handouts. Tufte's style is known for its extensive use of sidenotes, tight integration of graphics with text, and well-set typography. This style has been implemented in LaTeX and HTML/CSS

For org-mode HTML output, we can achieve similar results by:

Using custom CSS (similar to tufte-css) Leveraging org-mode's native margin note capabilities Using Python for data visualization instead of R

Margin Figures

Images and graphics play an integral role in Tufte's work. To place figures in the margin you can use org-mode's #+ATTR_HTML: with custom CSS classes:

You can adjust the proportions of figures using the figsize parameter in matplotlib. These are specified in inches, and will be automatically scaled down to fit within the handout margin through CSS.

Arbitrary Margin Content

In fact, you can include anything in the margin using org-mode's footnotes or custom HTML blocks:

#+begin export html <div class="margin-note"> We know from <em>the first fundamental theorem of calculus</em> that for <em>x</em> in [<em>a</em>, <em>b</em>]: <p>\[\frac{d}{dx}\left( \int_{a}^{x} f(u)\,du\right)=f(x).\]</p> </div> #+end export

For the sake of portability, you should keep the margin content as simple as possible. You may use simple org-mode syntax like bold and /italic/ text.

Main Column Figures

Besides margin and full width figures, you can of course also include figures constrained to the main column. This is the default type of figures in org-mode HTML output.

Sidenotes

One of the most prominent and distinctive features of this style is the extensive use of sidenotes. There is a wide margin to provide ample room for sidenotes and small figures. Any use of a footnote will automatically be converted to a sidenote.

If you'd like to place ancillary information in the margin without the sidenote mark, you can use custom HTML blocks or org-mode's margin note capabilities.

#+begin export html <div class="margin-note">This is a margin note. Notice that there is no number preceding the note.</div> #+end export

Tables

You can format tables that integrate well with the rest of the Tufte handout style. The table captions can be placed in the margin like figures in the HTML output with appropriate CSS.

Responsiveness

The HTML page can be made responsive through CSS media queries. When the page width is smaller than 760px, sidenotes and margin notes can be hidden by default, with JavaScript controls to toggle their visibility.
