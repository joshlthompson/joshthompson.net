This project converts Org footnotes to simple tufte style side notes for export to HTML and PDF, using Org-Exporter. The implementation of the side notes is in Lisp and the accompanying style sheets were inspired by Jez's Pandoc Markdown CSS Theme.

Features

You can make words bold, italic, underlined, verbatim and code, and, if you must, strike-through. Text in the code and verbatim string is not processed for Org specific syntax; it is exported verbatim. Org provides a single command as entry point for inserting the marker character.

I will add something for highlighting, soon. Typesetting is influenced by:

  • Tufte CSS
  • Butterick's something ro other.

Introduction

The Tufte handout style is a style that Edward Tufte used in his books and handouts. Tufte’s style is known for its extensive use of sidenotes, tight integration of graphics with text, and well-set typography. This style has been implemented in LaTeX and HTML/CSS11

  tufte::tufte_html: default

Maths

I don't say with my chest, but a relatively painless method to add maths to an org file is AucTex + cdlatex, potentially with the additions of abbreviations + abbrev-mode.

  1. enable the cdlatex minor mode with M-x org-cdlatex-mode
  2. enable org-toggle-pretty-entities
  3. enable org-latex-preview. You'll need AucTex for this..
  4. For bindings, Kathink's suggestion is to suggestion is bind F2 to preview-map:

(define-key LaTeX-mode-map (kbd "<f2>") preview-map)

Then you can access the preview commands with F2 C-p instead of C-c C-p C-p etc, and the preview map remains internally consistent.

Let

\[ A = \begin{bmatrix} 4 & 1 \\ 2 & 3 \end{bmatrix} \]

We want to find \( \lambda \) and \( \mathbf{v} \neq \mathbf{0} \) such that:

\[ A\mathbf{v} = \lambda \mathbf{v} \]

Solutions:

\[ \lambda_1 = 5, \quad \lambda_2 = 2 \]

Each eigenvalue has a corresponding eigenvector \( \mathbf{v} \) such that \( (A - \lambda I)\mathbf{v} = 0 \).
